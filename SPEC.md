# Спецификация: glavredus - VK Group Searcher

## 1. Обзор проекта

**Название:** glavredus  
**Тип:** Веб-приложение (SvelteJS SPA)  
**Основная функция:** Поиск групп в социальной сети ВКонтакте по ключевым словам, с возможностью фильтрации и просмотра детальной информации о группах.  
**Целевые пользователи:** Маркетологи, SMM-специалисты, владельцы бизнеса, ищущие сообщества для продвижения.

### Компоненты

1. **SearchInput**
   - Поле ввода с иконкой поиска
   - Placeholder: "Введите название группы..."
   - States: default, focused (голубая рамка), error (красная рамка)

2. **FilterSelect**
   - Выпадающие списки для фильтров
   - Типы групп: Все, Открытые, Закрытые
   - Сортировка: По умолчанию, По участникам, По росту

3. **GroupCard**
   - Аватар группы (круглый, 80px)
   - Название группы (ссылка)
   - Описание (2 строки max)
   - Участники (с форматированием: 1.5K, 15.3K, 1.2M)
   - Статус (открытая/закрытая)
   - Кнопка "Подробнее"

4. **LoadingSpinner**
   - Анимация загрузки
   - Центрированный спиннер

5. **ErrorMessage**
   - Красный блок с текстом ошибки

6. **EmptyState**
   - Иконка + текст при отсутствии результатов

## 3. Функциональная спецификация

### Основные функции

1. **Поиск групп**
   - Ввод ключевого слова
   - Запуск поиска по нажатию Enter или кнопки
   - Минимальная длина запроса: 2 символа

2. **Фильтрация**
   - По типу группы (открытая/закрытая)
   - По количеству участников

3. **Пагинация**
   - Загрузка по 20 групп за раз
   - Кнопка "Загрузить ещё"

4. **Детальный просмотр**
   - Модальное окно с полной информацией
   - Описание, статистика, ссылка на группу

### Стор (State Management)

```javascript
{
  query: string,
  groups: Group[],
  loading: boolean,
  error: string | null,
  filters: {
    type: 'all' | 'open' | 'closed',
    sort: 'default' | 'members' | 'growth'
  },
  selectedGroup: Group | null,
  offset: number,
  hasMore: boolean
}
```

### API (mock-данные)

Так как VK API требует авторизации, используем моковые данные для демонстрации. Структура данных:

```typescript
interface Group {
  id: number;
  name: string;
  screen_name: string;
  description: string;
  avatar: string;
  members_count: number;
  is_closed: boolean;
  is_verified: boolean | null;
  activity: string;
}
```
